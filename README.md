## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* Swiper.js
* Google Fonts

---

## 🎨 Funcionalidades

* Layout moderno e responsivo
* Adaptação para desktop, mobile e tablet
* Seção hero com destaque do produto
* Carrossel interativo de vídeos com Swiper
* Animações em CSS
* Navegação simples e intuitiva
